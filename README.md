# 📚 Smart Resource LMS

A Smart Learning Management System (LMS) built as a learning project to demonstrate
core Computer Science concepts through a real web application.

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript
- **Backend:** Python Flask
- **Database:** SQLite
- **File Storage:** Local file system

## 🧠 CS Concepts Demonstrated
- **DAA:** Merge Sort, Greedy Algorithm, Dynamic Programming, Cosine Similarity
- **OS:** Multithreading, FCFS & Round Robin Scheduling, File Management
- **DBMS:** Relational schema design, SQL queries, activity logging

## 🚀 Features
- Student login and dashboard
- Resource library with upload and download
- Smart resource recommendations
- Learning progress tracker
- OS scheduling simulation

## ▶️ Run Locally
```bash
git clone https://github.com/YOUR_USERNAME/smart-lms.git
cd smart-lms
python -m venv venv
venv\Scripts\activate
pip install flask
python app.py
```

Then open `http://127.0.0.1:5000` in your browser.
